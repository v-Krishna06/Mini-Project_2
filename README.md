# Photo Studio UI

A modern and responsive photography portfolio landing page built entirely with **HTML** and **CSS**. The project features a clean dark-themed interface, a fixed sidebar navigation, social media links, and a full-screen hero section, showcasing a minimalist and professional web design.

---

## Features

- Responsive sidebar navigation
- Full-screen hero background
- Clean and modern dark UI
- Social media section
- Font Awesome icons integration
- Pure HTML and CSS (No JavaScript)

---

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts
